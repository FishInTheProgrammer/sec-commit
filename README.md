# Element UI Admin

> 一个干净优雅的Element UI Admin模板



## Start

 - 克隆或者下载这个仓库
 - 进入项目目录安装依赖

``` bash
yarn
```

## Develop

``` bash
# serve with hot reload at localhost:8010
npm run dev
```

``` bash
# build for production with minification
npm run build
```
